# My learning of C
Repository for tracking my progression and my exercises.

## Progression
**Steps in C:**
1. Source code (.c) ==> Compiler
2. Object code (.obj / .o) ==> Linker (add libraries)
3. Executable (.exe / a.out) ==> Executing

## Important Notes
* **Command for compilation:** `gcc main.c -o my_program`
* **Command for executing:** `./my_program`
